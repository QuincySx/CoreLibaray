## 搭建项目万能核心库
 采取MVP+VM模式，应用Retrofit+Rxjava2等常用库
 如果想查看Dagger2+以上配置请查看Dagger分支

### 使用时 
```
添加 keystore.props.file=../keystore.properties 到local.properties文件
```
